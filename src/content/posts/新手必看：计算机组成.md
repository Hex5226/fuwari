## 前言：为什么你要知道这些？

不管你用的是笔记本还是台式机，里面的核心部件其实大同小异。知道它们各自干什么，你就不会在买电脑时被销售员牵着鼻子走，也不会在电脑变慢时只会拍桌子。而且……自己动手加个内存、换个固态，真的比换手机电池简单多了。

### 1. CPU（中央处理器）—— 电脑的大脑 <a id="cpu"></a>
负责所有计算和指令处理。你可以理解为工厂里那个出方案的总工程师，脑子越快，干活越利索。

---

### 基础知识：CPU型号怎么看？  
  ### 一、Intel 酷睿系列（Core）

#### 1. 品牌定位：i3 / i5 / i7 / i9

- **i3**：入门级，办公、看视频、简单编程够用。
- **i5**：主流级，适合大多数学生、轻度游戏、编程。
- **i7**：高端级，适合重度多任务、大型游戏、视频剪辑。
- **i9**：旗舰级，适合专业渲染、极致游戏、多核需求。

> **注意**：这只是定位，**同代内** i7 一定比 i5 强，但不同代之间 i5 可能吊打老 i7。

#### 2. 代际（若型号为四位数，则代际为第一个数，若型号为五位数，则代际为前两个数）

以 `i5-13500H` 为例：
- `13` 表示第 13 代。
- 代数越高，架构越新，性能越强（通常）。
- 目前主流是 12/13/14 代，2026 年已推出第三代酷睿 Ultra 系列（新命名体系）。

#### 3. 后缀（最后几个字母）

后缀决定了功耗和适用场景：

| 后缀 | 含义 | 常见设备 |
|------|------|----------|
| **U** | 低功耗（低压版） | 轻薄本、商务本 |
| **H** | 高性能（标压版） | 游戏本、全能本 |
| **HX** | 可超频高性能 | 顶级游戏本、工作站 |
| **K** | 可超频 | 台式机（需配 Z 系列主板） |
| **F** | 无核显 | 台式机（必须配独显） |
| 无后缀 | 标准版 | 台式机常规型号 |

**举例**：
- `i5-1235U`：12代低压版，常见于轻薄本。
- `i7-13700H`：13代标压版，游戏本主流。
- `i9-13900HX`：13代可超频旗舰，性能拉满。

---

### 二、AMD 锐龙系列（Ryzen）

#### 1. 品牌定位：R3 / R5 / R7 / R9

- **R3**：对标 i3
- **R5**：对标 i5
- **R7**：对标 i7
- **R9**：对标 i9

#### 2. 代际（第一个数字）

以 `R7-7840H` 为例：
- `7` 表示 7000 系列（第 7 代锐龙）。
- AMD 的命名规律不如 Intel 直观，但一般数字越大越新。

#### 3. 后缀

| 后缀 | 含义 |
|------|------|
| **U** | 低功耗（轻薄本） |
| **HS** | 低功耗高性能（轻薄全能本） |
| **H** | 高性能（游戏本） |
| **HX** | 可超频高性能（顶级游戏本） |

---


**常见误区**：
- “i7一定比i5好”？  
  不一定！要看代数和后缀。比如 i7-1260P（12代低压）可能还不如 i5-13500H（13代标压）跑得快。**看具体型号，别看i几**。

- “核心越多越快”？  
  不一定。很多软件根本用不到那么多核，单核性能更重要。打游戏主要看单核，渲染才看多核。

### 2. 内存（RAM）—— 短期记忆 <a id="ram"></a>
你正在处理的文档、开的网页、运行的程序，都临时存在这里。一旦关机，内存里的东西就清空。

**常见误区**：
- “内存就是存储空间？”  
  并不是，“存储空间”是 __存放文件软件等__ 的空间被称作存储空间，它是硬盘存放东西多少的一个量  
  而“内存”指的是电脑运行过程中 __存储正在运行的进程的数据__ 所用的空间  
    
    **举个例子**：植物大战僵尸这个游戏需要先下载到你的硬盘中，占用硬盘的 __存储空间__ ，而游戏内拾取的阳光或硬币的数量是 __存在内存中__ 的。   
    
    **通俗来讲**：一个是“长期记忆”（硬盘存储空间），另一个是“短期记忆”（内存），即使是手机也是同理，很多人将两者混为一谈，已经成为最典型的误区之一。
- “内存越大电脑越快”？  
  够用就行。8GB 现在有点紧张，16GB 是主流，32GB 对绝大多数人来说是过剩的。**别被“超大内存”忽悠，关键是看够不够你用**。

- “内存频率越高越好”？  
  理论上是的，但实际体验差别不大，尤其是你不超频的话。优先保证容量，再考虑频率。

### 3. 硬盘（存储）—— 长期记忆 <a id="storage"></a>
你的系统、游戏、照片、电影都存这里。现在主流是 **固态硬盘（SSD）**，千万别再买纯机械硬盘（HDD）的电脑，否则开机慢到你怀疑人生。

**常见误区**：
- “容量越大越好”？  
  容量当然大点好，但更重要的是速度。NVMe SSD 比 SATA SSD 快好几倍，优先选 NVMe 接口的，商家或线下店都会有标注这两类， __不用担心不会分辨__ ，都2026了，我建议直接选择NVMe M.2。 <a id="ssd-type"></a>

- “固态硬盘不用分区”？  
  分不分区看个人习惯，但建议系统盘单独分一个区（比如 C 盘 200GB），方便重装系统时不用备份全部数据。

### 4. 显卡（GPU）—— 图像处理中心 <a id="gpu"></a>
负责把画面渲染出来。打游戏、剪视频、搞设计，显卡很重要。不玩游戏的话，CPU 自带的核显足够。

## 显卡的“身份证”：品牌 + 系列 + 数字 + 后缀

市面主流显卡只有两家：**NVIDIA（英伟达）** 和 **AMD（超威）**。  
不管哪家，型号基本都遵循：  
`品牌系列 + 数字代号 + 字母后缀`  

比如 `RTX 4060`，`RX 6750 GRE`。  
  
>  - 需要注意的是，一些CPU自带显卡（核芯显卡）简称核显，通常核显性能一般，游戏玩家一定要选择独立显卡（独显），就像上述的英伟达和AMD <a id="gpu-type"></a>

### 基础知识：显卡型号怎么看
### 一、NVIDIA 

### 1. 前缀系列

| 前缀 | 含义 | 常见于 |
|------|------|--------|
| **GT** | 入门级，性能很弱 | 办公、亮机 |
| **GTX** | 中端/老架构（已逐渐淘汰） | 10系、16系 |
| **RTX** | 高端，支持光线追踪、DLSS | 20系、30系、40系、50系 |

**一句话**：现在买新卡，认准 **RTX**。

### 2. 数字代号

以 **RTX 4060** 为例：

- 第一位数字（4）→ **代际**：40 系列。数字越大，架构越新。
- 后两位（60）→ **定位**：60 是甜品级，数字越大性能越强（如 70 > 60 > 50）。

**代际对应年份**：
- RTX 20 系列：2018–2019
- RTX 30 系列：2020–2022
- RTX 40 系列：2022–2024
- RTX 50 系列：2025–2026（最新）

**定位规律**：
- 50/60：主流游戏卡（性价比高）
- 70：高端游戏卡
- 80/90：旗舰（4K 游戏、专业需求）

### 3. 后缀

| 后缀 | 含义 |
|------|------|
| **Ti** | 同型号增强版，比不带 Ti 强（如 4060 Ti > 4060） |
| **SUPER** | 中期改款，性能小幅提升（如 4070 SUPER） |
| **无后缀** | 标准版 |

**举例**：
- `RTX 3060 Ti`：30 系，甜品级加强版，比 3060 强。
- `RTX 4070 SUPER`：40 系，70 级中期升级款。

---

## 二、AMD 显卡怎么看

### 1. 前缀系列

| 前缀 | 含义 |
|------|------|
| **Radeon** | 全称，但一般不写在型号里 |
| **RX** | 主流游戏卡，目前所有新卡都用这个前缀 |

### 2. 数字代号

以 **RX 6750 GRE** 为例：

- 第一位数字（6）→ **代际**：6000 系列。7000 系列为最新。
- 后两位（50）→ **定位**：数字越大性能越强（如 6800 > 6700）。

**代际对照**：
- RX 5000 系列：2019–2020
- RX 6000 系列：2020–2022
- RX 7000 系列：2022–2026

**定位规律**：
- 50/60：主流入门
- 70：高端
- 80/90：旗舰

### 3. 后缀

| 后缀 | 含义 |
|------|------|
| **XT** | 同型号加强版（如 6700 XT > 6700） |
| **GRE** | 中国特供版，性价比高（如 6750 GRE） |
| **无后缀** | 标准版 |

**举例**：
- `RX 6600 XT`：6000 系，主流加强版。
- `RX 7900 XT`：7000 系，次旗舰。

**常见误区**：
- “独显一定比核显好”？  
  不一定。现在很多核显（比如 AMD 的 Radeon 680M/780M）已经能流畅玩网游了，而有些入门独显（比如 MX系列）还不如核显。**别盲目追求独显**。

- “显存越大显卡越强”？  
  显存只是显存，核心架构和频率更重要。比如 RTX 3050 给 8GB 显存也打不过 RTX 4060 的 6GB。**看型号，别看显存大小**。

### 5. 主板 —— 身体骨架 <a id="motherboard"></a>
把所有零件连接在一起，提供各种接口。它不直接影响性能，但决定了你能插什么 CPU、能装多大内存、有没有足够的接口。

**常见误区**：
- “主板越贵越好”？  
  不是。够用就行，只要供电能满足 CPU、接口够用、品牌靠谱，没必要追求顶级主板。**别在主板上面浪费预算**。

### 6. 电源 —— 心脏 <a id="psu"></a>
给所有零件供电。功率不够会导致电脑不稳定甚至烧坏硬件。

**常见误区**：
- “电源随便买个杂牌就行”？  
  绝对不行！劣质电源是电脑杀手。一定要选正规品牌（如海韵、振华、全汉、长城等），并且留出余量（比如你配置预估 400W，就买 550W 以上）。**电源千万别省**。

### 7. 散热器 —— 空调 <a id="cooler"></a>
CPU 和显卡发热，不散热就会降频（变慢）甚至烧坏。

**常见误区**：
- “水冷一定比风冷好”？  
  不一定。高端风冷（比如猫头鹰）比很多入门水冷还稳，还不用担心漏液。除非你玩超频，否则百元风冷足够压住 i5/R5 级别的 CPU。

## 二、几个常见“智商税”点

1. **“i7级” “i9级”**  
   这是淘宝整机最喜欢用的词。实际上可能是十年前的老架构，性能还不如现在的 i3。**凡是不写具体型号的，直接划走**。

2. **“超大显存游戏显卡”**  
   比如 GT 730 4G 版，显存大，但核心弱，打游戏照样卡。**看显卡型号，别只看显存**。

3. **“军工级主板”**  
   没有这回事，就是普通主板。**别为噱头多花钱**。

4. **“静音电源”**  
   噪音主要来自风扇，电源本身不发声。**不必迷信静音电源**。

5. **“笔记本独显直连”**
   这是个好功能，但不是必需品。如果常玩 FPS 游戏，可以优先选支持独显直连的机型；普通用户无所谓。

## 三、升级避坑指南

- **笔记本**：大部分笔记本只能换内存和硬盘，CPU 和显卡是焊死的。想升级的话，买之前就看准配置。
- **台式机**：升级很方便。但注意：
  - 加内存要买同代、同频率，最好同品牌。
  - 换显卡前先看看电源功率够不够。
  - 加硬盘前检查主板有没有多余的 M.2 插槽或 SATA 口。

## 四、我该从哪开始了解自己的电脑？

**看配置**：  
- 按 `Win + R`，输入 `dxdiag`，可以看到 CPU、内存、显卡信息。  
- 硬盘信息可以在“此电脑”右键 → 管理 → 磁盘管理里看到。

**测试性能**：  
- 跑分软件：Cinebench（测 CPU）、3DMark（测显卡），但不用太纠结分数，用着流畅就行。
